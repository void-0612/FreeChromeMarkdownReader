# Open Markdown Reader

一个 Manifest V3 浏览器插件，用来在 Chrome / Edge 里直接阅读 Markdown 文件，并免费提供目录侧栏。

## 功能

- 自动识别 `.md`、`.markdown`、`.mdown`、`.mkd` 页面
- 渲染标题、段落、链接、图片、代码块、引用、列表、任务列表和表格
- 左侧目录侧栏，可点击跳转
- 滚动时高亮当前目录项
- 可在“标题目录”和“文件目录”之间切换
- 文件目录会尝试读取当前 Web 目录；本地文件第一次需要点击“选择文件夹”授权，之后会自动恢复上次授权目录
- 支持编辑 Markdown；有文件夹授权的本地文件可保存回原文件
- 支持日间、夜间、纸张、森林、海蓝主题切换，并记住偏好
- 支持本地 Markdown 文件；需要在浏览器扩展详情页开启 `Allow access to file URLs`

## 安装

1. 打开 Chrome / Edge 扩展管理页面。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择本仓库里的 `extension` 文件夹。

## 测试

打开任意 Markdown 文件 URL，或直接在浏览器里打开本地 `.md` 文件。若是本地文件，请先在插件详情页开启文件 URL 访问权限。

## 切换 Markdown 文件

1. 打开一个 Markdown 文件。
2. 点击左侧栏上方的“文件目录”。
3. 如果文件目录没有自动出现，点击“选择文件夹”。
4. 选择当前 Markdown 所在文件夹，或选择其他文件夹。
5. 点击文件目录里的 `.md` / `.markdown` 文件即可切换阅读。

浏览器安全策略不允许插件静默扫描本地 `file://` 文件夹，所以本地文件第一次需要手动选择文件夹授权。授权会被浏览器记住；之后打开本地 Markdown 时，插件会自动恢复上次授权目录并显示文件目录。

## 编辑 Markdown

1. 点击阅读区顶部的“编辑”。
2. 修改 Markdown 内容。
3. 如果当前文件来自已授权的文件目录，点击“保存”会写回原文件。
4. 如果保存按钮不可用，请先在“文件目录”里点击“选择文件夹”，再从文件目录中打开要编辑的文件。
